### Development Setup
You should have `gulp-cli` installed on your system already: `yarn global add gulp-cli`.

```bash
# Install dependencies
yarn install

# Watch for changes to TypeScript, and build on change
gulp watch
```

### Testing
Note: test files should have the suffix: `.spec.ts`
```bash
# Run full test suite
npm test
```
